# 🧮 diaww Calculator - Premium Dark Edition
License: MIT

HTML5

CSS3

JavaScript
Kalkulator modern dengan desain **Dark Glassmorphism** yang elegan, responsif, dan interaktif. Proyek ini dibuat dengan fokus pada pengalaman pengguna (UX) yang halus dan tampilan UI yang kekinian.
## ✨ Fitur Utama
 * **Intro Animation**: Layar pembuka (*Welcome Screen*) eksklusif dengan teks animasi "diaww".
 * **Glassmorphism UI**: Desain transparan dengan efek blur latar belakang yang premium.
 * **Full Responsive**: Tampilan yang menyesuaikan secara otomatis untuk perangkat Mobile (HP) maupun Desktop.
 * **Audio Feedback**: Efek suara "klik" yang realistis setiap kali tombol ditekan (menggunakan Web Audio API).
 * **Dark Mode**: Skema warna Midnight Slate yang nyaman di mata untuk penggunaan jangka panjang.
 * **Dukungan Keyboard**: Bisa dioperasikan langsung menggunakan tombol angka dan simbol di keyboard komputer.
 * **Local Formatting**: Format angka sesuai standar Indonesia (titik ribuan dan koma desimal).
## 🚀 Pratinjau
> **Catatan**: Untuk melihat demo langsung, Anda bisa mengunggah file ini ke GitHub Pages atau membukanya langsung di browser Anda.

## 🛠️ Teknologi yang Digunakan
 * **HTML5**: Struktur semantik.
 * **CSS3**: Styling manual dengan variabel CSS, Flexbox, Grid, dan Animasi.
 * **JavaScript (Vanilla)**: Logika kalkulasi, Web Audio API, dan manajemen state.
 * **Google Fonts**: Menggunakan font 'Poppins' untuk tipografi yang bersih.

## 🎹 Cara Penggunaan
 * Klik tombol pada layar atau gunakan **Keyboard**:
   * 0-9 : Angka
   * . : Desimal
   * +, -, *, / : Operasi aritmatika
   * Enter atau = : Hasil
   * Backspace : Menghapus satu angka
   * Esc : Menghapus semua (AC)
